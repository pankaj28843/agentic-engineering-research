## GPT-5.6 ships, and a production migration tests the claim

OpenAI released GPT-5.6 Sol, Terra, and Luna through its API, ChatGPT, and Codex,
with multi-agent `ultra` mode. Input/output prices per million tokens were
$5/$30, $2.50/$15, and $1/$6. Ploy then migrated a production agent from Opus
4.8 and, across hundreds of fixtures, measured 2.2 times faster completion, 27%
lower cost, about half the output tokens, and a higher visual score. The
workload-specific test exposed hidden harness assumptions.

<div class="impact" markdown="1">

**Builder impact:** Evaluate a new model with production fixtures and audit tool schemas, caching, and token behavior before accepting either vendor or first-run cost claims.

</div>

<div class="caveat" markdown="1">

**Caveat:** Ploy's measurements apply to its agent and harness; OpenAI's broader benchmark claims remain vendor-reported.

### Meta opens Muse Spark 1.1 and its Model API

Meta put Muse Spark 1.1 into public preview with a one-million-token context,
multi-agent orchestration, computer use, and a Model API. An HN contributor
found its Terminal-Bench 2.1 comparison used six CPUs and 8 GB for every task
despite lower task-specific limits. Availability stands; the comparison does
not.

<div class="impact" markdown="1">

**Builder impact:** The new API surface is inspectable, but reproduce agent benchmarks under each task's official resource limits before selecting the model.

</div>

<div class="evidence" markdown="1">
