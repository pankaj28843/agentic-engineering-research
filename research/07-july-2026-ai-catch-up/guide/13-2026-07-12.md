## Claude Code sent 33,000 fixed tokens before the task

Systima logged requests at the API boundary while holding model, machine, and tasks constant. In its pinned setup, Claude Code sent about 33,000 fixed tokens before reading the prompt, versus about 7,000 for OpenCode. A 72 KB instruction file added roughly 20,000 tokens per request; five MCP servers added another 5,000 to 7,000; a 121,000-token direct task became 513,000 tokens with two subagents. The study separately reran reduced configurations and a ten-lane quality benchmark, making this component attribution more useful than a meter screenshot. It measures Claude Code 2.1.207 and OpenCode 1.17.18, not permanent product behavior.

<div class="impact" markdown="1">

**Builder impact:** Budget context at the harness boundary, inventory always-loaded instructions and tool schemas, and require subagents to justify their multiplicative cost.

</div>

<div class="discussion" markdown="1">

**Discussion:** Practitioners supplied cases where subagents exhausted budgets, alongside a real counterpoint: parallel exploration can buy quality when uncertainty warrants the extra context.
