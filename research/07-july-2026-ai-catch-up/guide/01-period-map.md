# Period Map: Six Changes That Matter

The first three weeks of July changed the engineering picture in six connected
ways. Use these arcs to choose where to enter the chronology.

### Evaluation became an engineering system

OpenAI found roughly 30% of SWE-Bench Pro public broken; Databricks showed that
private tasks, harness choice, and cost per solved task can reverse a buying
decision. APIFlow-Bench later showed agents completing the workspace work but
losing the correct final answer, while CrucibleBench showed one judge component
moving rankings by six places. Follow [2 July](03-2026-07-02.md),
[8 July](09-2026-07-08.md), and [22 July](23-2026-07-22.md).

### The harness became part of the model

Token anomalies, schema failures, hidden context overhead, question timeouts,
and context-allocation metadata changed outcomes without new model weights. By
[22 July](23-2026-07-22.md), Microsoft's released harness made the useful unit
explicit: model plus persistence, policy, context control, and telemetry.

### Agent scale exposed coordination costs

Vending simulations produced strategic behavior; Cursor's SQLite experiment
made conflicts, roles, and cost visible. From [6 July](07-2026-07-06.md) to
[20 July](21-2026-07-20.md), agent count raised orchestration work before useful
throughput.

### Human control moved into the safety boundary

AI-assisted audits found real bugs, but an autonomous-agent intrusion and a
Claude Code timeout made prompts, credentials, and escalation paths security
controls. Read [7 July](08-2026-07-07.md),
[16 July](17-2026-07-16.md), and [17 July](18-2026-07-17.md) as one arc.

### Open weights became deployment strategy

Kimi, Inkling, local runtimes, and Chinese open-weight distribution widened the
choice set. The useful distinction is which parts of inference, customization,
licensing, hardware, and operations a team controls. Start on
[15 July](16-2026-07-15.md).

### Economics shifted toward completed outcomes

Cheap tokens did not guarantee cheap work. Private evaluation, harness context,
security models, and mixed-model swarms pointed toward cost per accepted result,
including review and recovery. Read [12 July](13-2026-07-12.md) and
[21 July](22-2026-07-21.md).

**Fastest reading route:** read 8, 14, 16, 17, 20, and 22 July, then use
[What to Carry Forward](24-what-to-carry-forward.md) to turn the reporting into
an operating checklist.
