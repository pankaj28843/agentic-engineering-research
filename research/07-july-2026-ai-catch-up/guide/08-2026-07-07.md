<article class="lead-story" markdown="1">

## AI-assisted audit finds seven real CIRCL bugs


zkSecurity reported seven zkao-assisted bugs in Cloudflare's CIRCL library,
including a CP-ABE access-control break. Researchers validated candidates,
built proofs of concept, and linked exact upstream fixes; most findings received
HackerOne bounties. That chain is stronger than a raw issue list, although
severity labels still need care.

HN readers focused on false positives and human validation: AI expands search;
experts establish exploitability and coordinate repair.

<div class="impact" markdown="1">

**Builder impact:** AI-assisted audits should preserve candidate-to-PoC-to-fix
lineage, budget explicitly for expert triage, and report rejected candidates so
false-positive cost is visible.

</div>

<div class="caveat" markdown="1">

**Caveat:** Seven fixed bugs establish value for this audit, not a general
detection rate or proof that the agent can validate cryptography independently.

</div>

### Riddle turns handwriting into an agent interface


Riddle is an open-source Rust application that converts reMarkable pen strokes
and streams an LLM's handwritten reply. It supports an OpenAI-compatible key or
Pi and documents root/device takeover risk. July 7 comes only from HN discovery;
the repository is mutable.

<div class="impact" markdown="1">

**Builder impact:** Device-native agent interfaces need explicit boundaries for
capture, memory, remote-provider use, and privileged installation, especially
when an ambient notebook becomes an input channel.

</div>

<div class="caveat" markdown="1">

**Caveat:** This is an inspectable demonstration, not evidence of production
reliability, privacy, or a stable July 7 release state.

</div>
